# Portfolio-Data-Science
# Análisis de evasión de clientes y limpieza de datos paraTelecom X.
Este proyecto analiza los factores que influyen en la pérdida de clientes (churn) para una empresa de telecomunicaciones, utilizando técnicas de **ETL**, **Limpieza de Datos** y **Análisis Exploratorio (EDA)**.

## Resumen del Proyecto
El objetivo principal es identificar patrones de comportamiento en los clientes que abandonan el servicio para proponer estrategias de retención.

###  Principales Hallazgos (Insights)
* **Efecto Ecosistema:** Los clientes con más servicios adicionales contratados tienen una tasa de evasión significativamente menor.
* **Sensibilidad al Precio:** El cargo diario impacta directamente en la decisión de permanencia.
* **Lealtad Inicial:** Los primeros meses de contrato son críticos para la retención.

## Tecnologías Utilizadas
* **Python** (Pandas, Seaborn, Matplotlib)
* **Google Colab** para el desarrollo.
* **JSON Normalization** para el procesamiento de datos anidados.

##  Visualizaciones Clave
El proyecto incluye análisis de correlación y gráficos de barras que muestran la "escalera de lealtad" según servicios contratados.
